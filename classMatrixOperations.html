Detailed Description

template<typename T>
class MatrixOperations< T >

Utility class containing basic matrix arithmetic operations.

This class provides static methods for:

  • Addition
  • Subtraction
  • Scalar multiplication

It acts as a service layer separating arithmetic logic from UI and algorithm strategy.
