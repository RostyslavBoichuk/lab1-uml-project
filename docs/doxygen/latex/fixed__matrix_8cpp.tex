\doxysubsection{Detailed Description}
Object-\/oriented matrix calculator with multiple multiplication strategies. 

This is the improved architectural version of the matrix application.

Features\+:
\begin{DoxyItemize}
\item Generic \doxylink{class_matrix}{Matrix} class (template-\/based)
\item Encapsulated matrix operations (add, subtract, scalar multiply)
\item Strategy pattern for multiplication (Classical / Strassen)
\item MVC-\/like separation (\doxylink{class_menu}{Menu} = View, \doxylink{class_controller}{Controller} = Application logic)
\item Smart pointer usage for polymorphism
\end{DoxyItemize}

The system is designed for\+:
\begin{DoxyItemize}
\item Unit testing
\item Extensibility
\item Algorithm comparison
\item Performance evaluation
\end{DoxyItemize}
