# Glossary – Matrix Calculator Project

## Matrix
A mathematical data structure consisting of rows and columns used to store numeric values.

## Matrix Operation
An operation performed on matrices such as addition, subtraction, scalar multiplication, transpose, or multiplication.

## Matrix Addition
Element-wise addition of two matrices of the same dimensions.

## Matrix Subtraction
Element-wise subtraction of two matrices of the same dimensions.

## Scalar Multiplication
Multiplying every element of a matrix by a scalar value.

## Matrix Multiplication
A mathematical operation where rows of the first matrix are multiplied by columns of the second matrix.

## Transpose
An operation that swaps rows and columns of a matrix.

## Multiplier Strategy
An algorithm abstraction used to perform matrix multiplication. Implemented via the `IMatrixMultiplier` interface.

## Classical Multiplier
Implementation of matrix multiplication using the standard O(n³) algorithm.

## Strassen Multiplier
Optimized matrix multiplication algorithm that reduces computational complexity using divide-and-conquer.

## Controller
Component responsible for handling user input and coordinating system operations.

## Menu
User interface component that allows selection of operations.

## Unit Test
Automated test that verifies correctness of system functionality using the Catch2 framework.
