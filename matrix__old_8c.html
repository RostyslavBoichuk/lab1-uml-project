Detailed Description

Legacy procedural implementation of a Matrix Calculator.

This file contains the original implementation of the matrix calculator using static 10x10 arrays and global loop variables.

It demonstrates procedural programming style before refactoring into an object-oriented design.

Warning
This is legacy code and contains poor architectural practices such as global variables and fixed-size arrays.
